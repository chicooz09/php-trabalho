História do PHP

O PHP nasceu em 1994, criado por Rasmus Lerdorf como um conjunto de scripts CGI chamado "Personal Home Page Tools". Inicialmente, era usado apenas para rastrear visitas ao seu currículo online. Com o tempo, a ferramenta evoluiu e ganhou recursos de banco de dados e formulários.

1994

Nascimento

Rasmus Lerdorf cria o PHP/FI (Personal Home Page / Forms Interpreter) como scripts CGI em C.

1997-1998

PHP 3

Andi Gutmans e Zeev Suraski reescrevem o núcleo do PHP, criando o Zend Engine. O nome muda para "PHP: Hypertext Preprocessor".

2000

PHP 4

Lançamento do Zend Engine 1.0, melhorando performance e recursos. PHP se torna uma das linguagens mais populares da web.

2004

PHP 5

Introduz Programação Orientada a Objetos completa (POO), exceções, e o Zend Engine 2.0.

2015

PHP 7

Grande salto de performance (2x mais rápido), tipos de retorno, e redução de consumo de memória.

2020-2024

PHP 8.x

JIT Compiler, Union Types, Named Arguments, Attributes (Annotations), Enums, e melhorias contínuas de performance.

Back-end, Front-end e Full Stack

Back-end

PHP é uma linguagem back-end!

Responsável pela lógica do servidor, processamento de dados, banco de dados, autenticação e APIs.

  • Processa requisições HTTP
  • Conecta com bancos de dados
  • Gerencia sessões e cookies
  • Implementa regras de negócio
  • Cria APIs REST/GraphQL

Front-end

Interface visual que o usuário vê e interage. PHP pode gerar HTML dinamicamente!

  • HTML (estrutura)
  • CSS (estilo e design)
  • JavaScript (interatividade)
  • React, Vue, Angular
  • PHP renderiza templates

Full Stack

Desenvolvedor que domina tanto front-end quanto back-end, capaz de criar aplicações completas.

  • PHP (back-end)
  • HTML/CSS/JS (front-end)
  • Banco de dados
  • APIs e integrações
  • Deploy e DevOps

Como PHP se Integra

No desenvolvimento web moderno, o PHP atua principalmente no back-end, processando dados e gerando respostas. Ele pode ser usado de duas formas principais:

1. Renderização Server-Side (Tradicional)

PHP gera o HTML completo no servidor e envia para o navegador. Usado em WordPress, Laravel com Blade, sistemas legados.

<?php echo "<h1>Olá, " . $nome . "</h1>"; ?>

2. API Backend (Moderno)

PHP cria APIs que retornam JSON. O front-end (React, Vue, Angular) consome esses dados e renderiza a interface. Arquitetura moderna e desacoplada.

// PHP retorna JSON
echo json_encode(['nome' => 'João', 'idade' => 25]);
